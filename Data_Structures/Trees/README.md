## Challenge
Implement a **Binary Tree** and a **Binary Search Tree** that does the
following following traversals:
1. PreOrder()
2. InOrder()
3. PostOrder()
4. BreadthFirst()

Allow the **Binary Tree** to Search().

Allow the **Binary Search Tree** to Search() and **AddNode()**.
