# Merge 2 Binary Trees
Problem from [here](https://leetcode.com/problems/merge-two-binary-trees/description/)

#### Input
```
                Tree 1                      Tree 2
                 11                           2
                 / \                         / \
                3   2                       1   6
               /                             \   \
              5                               9   7

```


#### Output
```
                13
               / \
              4   8
             / \   \
            5   9   7  

```
