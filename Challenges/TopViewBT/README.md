# Top View of a BT

Problem from [here](https://www.geeksforgeeks.org/print-nodes-top-view-binary-tree/), 
but the solution from [here](https://algorithms.tutorialhorizon.com/print-the-top-view-of-a-binary-tree/) 
makes the most sense.

#### Input
            1
           / \
          2   3
           \
            4
             \
              5
               \
                6

#### Output
2, 1, 3, 6
